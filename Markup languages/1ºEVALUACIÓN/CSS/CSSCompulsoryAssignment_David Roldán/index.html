HTML

HTML, acronym of HyperText Markup Language (Lenguaje de Marcado de Hipertexto), is the predominant markup language for the development of web pages. Is used for describing the structure and the content in form of text, as well as to complete the text with objects such as images. HTML is written in form of "tags", surrounded by angle brackets (<,>). HTML can also describe, until some point, the apparence of a document, and it can include a script (for example Javascript), which can affect the browsers' behaviour and other HTML processors.

HTML can also be used for referring to content of the type MIME text/html or even more widely as a generic term for the HTML, either in XML descended form (such as XHTML 1.0 and later) or in a descended form directly from SGML (as HTML 4.01 and previous).
